# Git Knowledge Assessment

This repository contains the setup for the GitHub Pages blog as part of the Dot Campus assignment, along with the completed Git Knowledge Assessment.

## Assessment Answers

1. **What is Git?**
   - A version control system

2. **Git is another name for GitHub?**
   - False

3. **What is the command to get the current status?**
   - `git status`

4. **Git automatically adds new files to be tracked/staged?**
   - False

5. **What is the command to commit staged changes?**
   - `git commit`

6. **What is the command to view history?**
   - `git log`

7. **What commands can be used to create a new branch named "feat/signup-page"?**
   - `git branch feat/signup-page` and `git checkout -b feat/signup-page`

8. **What is the command to delete the branch "chore/remove-error-log"?**
   - `git branch -d chore/remove-error-log`

9. **What is the command to clone the remote git repo?**
   - Use `git clone` followed by the repository URL provided in the question.

10. **"git pull" command is a combination of:**
    - fetch and merge
